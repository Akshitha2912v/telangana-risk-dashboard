# Telangana Risk Intelligence Dashboard

## 📊 Overview
This project identifies high-risk districts in Telangana by integrating rainfall, accident, and healthcare data. It helps highlight areas that require attention using a data-driven approach.

---

## 🚀 Features
- Data cleaning and preprocessing using Python (Pandas)
- Multi-factor Risk Index (0–100)
- Risk classification (High, Medium, Low)
- Recommendation system for decision-making
- Interactive dashboard built in Power BI

---

## 🛠 Tools Used
- Python (Pandas)
- Power BI

## 💡 Key Insight
Certain districts consistently show higher risk patterns, indicating the need for improved infrastructure, road safety, and healthcare facilities.

## ▶️ How to Run
1. Run Python scripts in the `scripts/` folder
