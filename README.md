# Telangana Risk Intelligence Dashboard

## 📊 Overview
This project identifies high-risk districts in Telangana using rainfall, accident, and healthcare data.

## 🚀 Features
- Data cleaning using Python (Pandas)
- Risk Index (0–100) calculation
- Risk classification (High, Medium, Low)
- Recommendation system
- Interactive Power BI dashboard

## 🛠 Tools Used
- Python (Pandas)
- Power BI

## 💡 Key Insight
Some districts consistently show higher risk, indicating the need for improved infrastructure and healthcare support.
